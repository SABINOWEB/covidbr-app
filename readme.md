## Descrição
Aplicativo criado com o intuito de trazer informação para a população sobre a propagação do vírus Covid-19.

## `npm` scripts

### Desenvolvimento
```sh
yarn dev # ou npm run dev
```

### Produção
```sh
yarn build # ou npm run build
```
